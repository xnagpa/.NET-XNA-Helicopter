=Helicopter
Небольшая игра на XNA. Играем за вертолёт, сбиваем врагов. 
Для выстрелов вертолета и дыма танков используется генератор частиц.
Симпатичная помощница прилагается ;)
Скриншоты находятся в папке game screens

p.s. 
Игра компилировалась под x86. Для запуска проект необходимо пересобрать. 
Может понадобаться XnaRedistributable.
